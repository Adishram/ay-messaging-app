ECDH Key Exchange

Using Elliptic-curve Diffie–Hellman, keys are generated on-device. The private key never leaves your hardware, ensuring perfect forward secrecy.

security_update_good

AES-256-GCM Encryption

Every message is encrypted using AES-256 with Galois/Counter Mode, providing both high speed and authenticated encryption for data integrity.

P2P WebRTC Infrastructure

Audio and video calls use peer-to-peer WebRTC streams, bypasssing central servers and ensuring the lowest possible latency without monitoring.

async function establishSecureLink(peerID) {
  const ephemeralKey = await crypto.subtle.generateKey(
    { name: "ECDH", namedCurve: "P-256" },
    false,
    ["deriveKey"]
  );
  
  // Direct P2P negotiation via WebRTC
  const connection = new RTCPeerConnection(config);
  
  // Only public keys are shared
  const secureChannel = await deriveSharedSecret(
    ephemeralKey.privateKey, 
    peerID.publicKey
  );
  
  return encryptSession(secureChannel);
}
